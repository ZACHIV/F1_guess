# F1 Guess MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a mobile-first F1 circuit guessing MVP with a single-screen interface and a local audio extraction CLI.

**Architecture:** Use a small Vite frontend with challenge data modules and utility helpers. Pair it with a Node CLI that shells out to `yt-dlp` and `ffmpeg`, writing extracted clips into the frontend public assets directory.

**Tech Stack:** Vite, vanilla JavaScript, CSS, Vitest, WaveSurfer.js, Node child process APIs
